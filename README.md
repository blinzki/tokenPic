# TokenPic scripts

Some scripts for TokenPic process



### Dependencias 

```
$ go get github.com/
```

### Install

```
$ go get github.com/blinzki/tokenpic
$ cd ~/go/src/github.com/blinzki/tokenpic/collection
$ go build
```
### Run 

```
$ ./collection
```
### Steps 
```
1. Put images in pic-in
2. Run collection
3. See result in pic-out folder
```
